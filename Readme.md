# ARCK Job Registry

Adds a simple dialogue allowing station managers to provide a list of NPCs available for hire on their station.

All stations are concerned, including player-owned stations.

Hiring skilled pilots, managers or boarding team isn't easy when starting the game, you can hunt for seminars or spend hours walking through stations and checking all NPCs you meet.

Some mods make seminars available from station traders, if the solution is interesting, it's somehow more like a cheat code.

This mod offer an in between : the station manager will provide you the list of all hireable NPCs on the station, you will be limited to the vanilla offer, but it will save you a lot of time. Finding a three star pilot should still be difficult as they are pretty rare, but recruiting a nice boarding team should be way easier.
